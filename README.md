# React Blog
Un frontend (muy simple) que consume la API [JSONPlaceholder](https://jsonplaceholder.typicode.com/). Desarrollado con React y [react-bulma-components](https://www.npmjs.com/package/react-bulma-components).

Para ejecutar localmente, clonar el repositorio, instalar dependencias con `npm install`, y ejecutar el servidor de desarrollo con `npm run dev` (o usar `npm run build` para crear una build de producci√≥n)
